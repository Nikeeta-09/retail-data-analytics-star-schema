# Retail Business Intelligence Dashboard

## Overview
This project focuses on building a complete retail data analytics and business intelligence solution using Data Warehousing concepts, SQL-based ETL processing, Star Schema modelling, and Power BI dashboards.

The system transforms raw retail transaction data into structured analytical insights for sales, customer behaviour, and business performance analysis.

---

## Features

- Retail data cleaning and transformation
- SQL-based ETL workflow
- Star Schema implementation
- Fact and Dimension table creation
- Power BI dashboard development
- Customer segmentation analysis
- Sales trend analysis
- Feedback and sentiment analysis
